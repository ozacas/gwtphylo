Java won't start?
Check that you have java 7 or later installed into your browser. In the URL box, you should see something like the image below when you put 'about:plugins' (without the quotes) into the URL box:

Remember that many browsers are only 32-bit even when installed on a 64-bit system, in which case you will need to install 32-bit java to match the browser. Be sure to install java with the browser closed, or results may be unpredictable.

Cannot see protein domains?
In applet mode, the default size for the protein domains is too small to see. You need to push the '+' button under the 'protein domains' section until the domains become visible.

Branch widths not viewable?
Not all of the available trees have branch widths to aid in visualisation of the major topologies. Consult the Methods page for each gene family for more details
